# My Tic-Tac-Toe

[README](README.md) | [中文文档](README_ZH.md)

## 介绍

这是一个使用[Godot](https://github.com/godotengine/godot)开源游戏引擎编写的井字棋游戏。

该项目作为作者学习Godot引擎的一个练手项目，属于入门级项目。
